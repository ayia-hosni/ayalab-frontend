Three-Pointer In-Place Reversal
⏱ O(n) time 📦 O(1) space 🔁 1 pass

Walk the list once, flipping each arrow backward with three variables: prev (the growing reversed portion), curr (the node being processed), and next (a temporary save so you never lose the rest of the chain).

1 next = curr.next Save before breaking
2 curr.next = prev Flip the arrow
3 prev = curr; curr = next Slide forward
