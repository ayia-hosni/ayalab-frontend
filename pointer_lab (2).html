Reverse Linked List

Level 206 · Recursion & Pointers

Given the head of a singly linked list, reverse the list and return the new head! 🔄

📋 Example 1
Input: head = [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
📋 Example 2
Input: head = [1, 2]
Output: [2, 1]
Constraints: Nodes: 0 to 5000 Value: -5000 to 5000
Hint 1 — The Three Pointers

Walk the list with two live pointers, prev and curr. From curr's point of view, its next arrow should point backward at prev instead of forward.

You also need a third, temporary pointer to remember the node ahead before you overwrite the link — otherwise the rest of the list is lost in space! 🚀

Hint 2 — The Loop Action Plan

1. next = curr.next — Save what comes after.

2. curr.next = prev — Flip the arrow around! ↩️

3. prev = curr — Slide prev forward.

4. curr = next — Slide curr forward.

When curr falls off the end, prev is your shiny new head.
